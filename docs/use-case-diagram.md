# Use Case Диаграмма — T-Путешествия (tbank.ru/travel)

```plantuml
@startuml
left to right direction
skinparam packageStyle rectangle
skinparam shadowing false

actor "Гость" as Guest

rectangle "T-Путешествия (tbank.ru/travel)" {

  usecase "UC-01: Просмотр главной страницы" as UC01
  usecase "UC-02: Переход между разделами" as UC02

  usecase "UC-03: Поиск авиабилетов" as UC03
  usecase "UC-04: Поиск отелей" as UC04
  usecase "UC-05: Поиск ЖД-билетов" as UC05
  usecase "UC-06: Просмотр раздела Туры" as UC06

  usecase "UC-07: Обмен городов местами (swap)" as UC07
  usecase "UC-08: Выбор даты вылета (календарь)" as UC08
  usecase "UC-09: Выбор пассажиров и класса" as UC09
  usecase "UC-10: Автодополнение городов" as UC10
}

Guest --> UC01
Guest --> UC02
Guest --> UC03
Guest --> UC04
Guest --> UC05
Guest --> UC06

' <<extend>>: расширяют сценарий поиска авиабилетов
UC07 ..> UC03 : <<extend>>
UC08 ..> UC03 : <<extend>>
UC09 ..> UC03 : <<extend>>

' <<include>>: автодополнение городов входит в каждый поиск
UC03 ..> UC10 : <<include>>
UC04 ..> UC10 : <<include>>
UC05 ..> UC10 : <<include>>

' <<extend>>: переход между разделами расширяет просмотр главной
UC02 ..> UC01 : <<extend>>
@enduml
```

## Акторы

| Актор | Описание                                                                       |
|-------|--------------------------------------------------------------------------------|
| Гость | Неавторизованный пользователь сайта. Просматривает разделы и выполняет поиски. |
