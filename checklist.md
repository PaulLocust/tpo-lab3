## UC-03 — Поиск авиабилетов (`FlightSearchTest`)

**Основной поток:**
- [x] Форма поиска содержит все ключевые элементы (Откуда, Куда, дата, swap, кнопка поиска)
- [x] Ввод в «Откуда» открывает выпадающий список подсказок
- [x] Выбор подсказки заполняет поле «Откуда»
- [x] Полный сценарий «Москва → Сочи → Найти билеты» открывает страницу выдачи рейсов

**Краевые случаи:**
- [x] Поиск с пустой формой не приводит к выдаче (валидация на клиенте)
- [x] Заполнено только «Откуда» (без «Куда») — поиск не идёт
- [x] Заполнено только «Куда» (без «Откуда») — поиск не идёт
- [x] Одинаковые города (Москва → Москва) — поиск блокируется
- [x] Очистка поля «Откуда» опустошает значение
- [x] Ввод заведомо несуществующего города не даёт реальных подсказок
- [x] Ввод одной буквы обрабатывается безопасно (форма остаётся живой)
- [x] Повторный набор города после очистки снова открывает подсказки
