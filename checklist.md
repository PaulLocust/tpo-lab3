# Чек-лист тестового покрытия — T-Путешествия (tbank.ru/travel)

Покрытие построено на основе 6 базовых прецедентов использования
(см. [docs/use-cases.md](docs/use-cases.md)) и их `<<include>>` / `<<extend>>`
прецедентов. Тестируется только UI: фильтры, отображение информации
в карточках, переходы. Никакого взаимодействия с формами, требующими
персональных данных.

---

## UC-1. Выбор авиабилетов (`FlightSearchTest`)

End-to-end сценарий: маршрут → дата → найти → применить фильтры →
проверить выдачу.

**Основной поток:**
- [x] End-to-end путь поиска: Откуда → Куда → дата → Найти → открывается выдача рейсов

**`<<include>>` шаги:**
- [x] UC-7: ввод города в «Откуда» открывает список подсказок, выбор подсказки заполняет поле
- [x] UC-7: ввод города в «Куда» открывает список подсказок, выбор подсказки заполняет поле
- [x] UC-8: клик по полю даты открывает виджет календаря, выбор ячейки заполняет дату

**`<<extend>>` фильтры в выдаче:**
- [x] UC-9 «Без пересадок»: каждая карточка содержит ровно один сегмент (две точки маршрута)
- [x] UC-9 «1 пересадка»: каждая карточка содержит как минимум один промежуточный пункт
- [x] UC-10 Сортировка «Сначала дешевле»: цена каждой следующей карточки `>=` цены предыдущей
- [x] UC-11 Фильтр авиакомпании: в каждой карточке упоминается выбранная авиакомпания
- [x] UC-12 Фильтр «С багажом»: в каждой карточке есть индикатор включённого багажа
- [x] UC-13 Фильтр времени вылета: время вылета в карточке попадает в выбранный диапазон

**Переход на покупку (ассоциация с UC-1):**
- [x] UC-21 «Выбрать» → «Продолжить» открывает страницу `/travel/flights/checkout/`

## UC-2. Выбор отеля (`HotelSearchTest`)

**Основной поток:**
- [x] End-to-end путь поиска: город → даты → Искать → открывается выдача отелей
- [x] Клик по карточке отеля открывает страницу отеля

**`<<include>>` шаги:**
- [x] UC-14: ввод города показывает выпадающий listbox подсказок, выбор заполняет поле
- [x] UC-8: даты заезда/выезда заданы (по умолчанию или вручную)

**`<<extend>>` фильтры в выдаче:**
- [x] UC-15 Сортировка «Сначала дешевле»: цена каждой следующей карточки `>=` цены предыдущей
- [x] UC-16 Фильтр «Спецпредложения»: в карточках присутствует индикатор скидки / процента
- [x] UC-17 Фильтр «Бесплатная отмена»: в карточках присутствует индикатор бесплатной отмены
- [x] UC-18 Добавление в избранное: после клика по «сердечку» отель появляется в разделе избранного

**Переход к бронированию (ассоциация с UC-2):**
- [x] UC-22 «Выбрать номер» в карточке отеля открывает страницу `/travel/hotels/new/checkout`

> Гостевые дома фильтр не проверяется (визуально не отличается в выдаче).
