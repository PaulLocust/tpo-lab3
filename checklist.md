## UC-02 — Переход между разделами сайта (`NavigationTest`)

**Основной поток:**
- [x] Клик «Авиа» открывает форму поиска авиабилетов
- [x] Клик «Отели» открывает форму поиска отелей
- [x] Клик «Поезда» открывает форму поиска ЖД-билетов
- [x] Клик «Туры» открывает раздел туров

## UC-03 — Поиск авиабилетов (`FlightSearchTest`)

**Основной поток:**
- [x] Форма поиска содержит все ключевые элементы (Откуда, Куда, Когда, swap, кнопка)
- [x] Ввод в «Куда» открывает выпадающий список подсказок
- [x] Выбор подсказки заполняет поле «Куда»
- [x] Полный сценарий «Москва → Сочи → Найти билеты» открывает страницу выдачи рейсов
- [x] Поле «Откуда» имеет дефолтное значение (Санкт-Петербург)

**Краевые случаи:**
- [x] Оба поля пустые → поиск не идёт
- [x] Заполнено только «Откуда» (без «Куда») — поиск не идёт
- [x] Очистка поля «Откуда» опустошает значение
- [x] Ввод бессмыслицы — ни одна подсказка не содержит введённой строки
- [x] Повторный набор города после очистки снова открывает подсказки
- [x] Выпадашка автокомплита содержит ≥ 1 пункта для известного города

- [x] Очистка поля города опустошает значение
- [x] Повторное открытие страницы не ломает форму

**Краевые случаи:**

## UC-07 — Дополнительные опции формы авиа (`ComplexRouteTest`)

На реальной форме T-Travel свопа городов нет; вместо него тестируем кнопку
«Сложный маршрут» и тоггл «Лечу по работе».

**Основной поток:**
- [x] Кнопка «Сложный маршрут» видна на форме поиска
- [x] Тоггл «Лечу по работе» присутствует в форме
- [x] Тоггл «Лечу по работе» переключается

**Краевые случаи:**
- [x] Клик «Сложный маршрут» не ломает форму поиска
- [x] Двойное переключение тоггла возвращает в исходное состояние
